# 优秀的 GPT 工具集

本列表旨在帮助开发者发现灵感，并快速上手开发基于 AI 的应用程序。

## 通用聊天机器人工具包

- **[chatbot-ui](https://github.com/mckaywrigley/chatbot-ui)**：开源的 ChatGPT 用户界面。
- **[nextjs-chatgpt-app](https://github.com/enricoros/nextjs-chatgpt-app)**：一个响应式聊天应用，基于 OpenAI 的技术。
- **[OpenChatKit](https://github.com/togethercomputer/OpenChatKit)**：强大的开源工具包，可用于创建多功能或专用聊天机器人，适合各种应用场景。
- **[OpenAssistant](https://github.com/LAION-AI/Open-Assistant)**：一个任务驱动的聊天助手，可以动态交互第三方系统并检索信息。

## 网页用户界面工具

- **[text-generation-webui](https://github.com/oobabooga/text-generation-webui)**：基于 Gradio 的 Web UI，可运行大型语言模型。

## 开发工具

- **[aicommits](https://github.com/Nutlope/aicommits)**：命令行工具，利用 AI 为你的 Git 提交生成消息。
- **[happycommit](https://github.com/jackbackes/happycommit)**：通过 Homebrew 安装的工具，支持生成任意大小的 Git 提交信息。

## 应用程序

- **[ShareGPT](https://github.com/domeccleston/sharegpt)**：一键分享你最有趣的 ChatGPT 对话。
- **[twitterbio](https://github.com/Nutlope/twitterbio)**：使用 ChatGPT 和 Vercel Edge Functions 生成你的 Twitter 个人简介。
- **[news-summarizer](https://github.com/Nutlope/news-summarizer)**：利用 AI 为新闻文章生成简短摘要。
- **[paul-graham-gpt](https://github.com/mckaywrigley/paul-graham-gpt)**：一个 AI 搜索与聊天工具，覆盖 Paul Graham 的所有文章。
- **[clarity-ai](https://github.com/mckaywrigley/clarity-ai)**：简单的 Perplexity AI 克隆版。
- **[notionGPT](https://github.com)**：将 Notion 与 ChatGPT 相结合的工具（链接需补充）。
